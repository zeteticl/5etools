使用 Foundry?

不？ 好吧，你应该试试！ Foundry 是现代的，比 Roll20 更好的替代品，更注重 Mod 的支持。它是 5etools 的首选的整合平台。因此，betterR20 将不再接受除了基础维护以外的任何更新。取而代之，开发支持已完全投入到为 Foundry 创建一套扩展工具，即为 Plutonium 模组。

安装此模组，复制清单文件地址，并在 Foundry 的 MOD 安装对话框里粘贴。查看 Plutonium 功能页面，以了解该 Mod 提供的功能的概述。

需注意的是，本站中文内容目前尚未支持在 Plutonium 模组于 Foundry VTT 中的使用。如需此功能，可以考虑加入 FVTT-CN/5etools 汉化组来贡献工作，尽早实现此功能。

使用 Roll20?

请注意： 5etools 的 Roll20 整合， betteR20，已经不再活跃开发。取而代之，开发支持已经完全投入到 Plutonium，如上文所述。

安装 Roll20 增强套件 以及 Tampermonkey (Chrome/Edge Chromium/Firefox)，并且点击链接获取 betteR20 脚本

作为玩家，你可以导入角色选项、法术以及物品来快速填充的角色卡。作为 GM，你可以额外导入怪物、表格以及冒险。

为了能够导入怪物，你的游戏必须拥有默认角色卡。我们强烈推荐在导入时使用 OGL，然后如果需要的话在之后切换到其他角色卡。

查看百科上的安装指南，以及可以查看帮助页来了解使用该脚本的一些基础用法。

FAQ

5etools 有百科吗？ 这里（英文）。 欢迎贡献内容。

5etools 免费吗？ 是的！此外，你还可以在 get.5e.tools 下载源代码。查看 自述文件 README 或者百科以获取帮助。

我可以为 5etools 英文站点贡献内容吗？ 源代码的拷贝在 GitHub 上公开。注意，这是一个镜像，所以并不保证内容始终最新。

我可以为 5etools 中文站点（正在查看的此站点）贡献内容吗？ 源代码的拷贝在 GitHub 上公开，可以直接为其提供 PR。或者，你可以通过果园的 5etools 翻译招募贴来加入 FVTT-CN 的 5etools 翻译组。

浏览器与设备支持？ 最新的桌面版 Chrome/Edge Chromium/Firefox。

怎么管理我自己的数据？ 并没有账号系统一说。一切都储存在 cookies 当中。如果你清除这些 cookies，或者关闭你的隐私模式窗口，储存的内容就会永久消失。为了安全保存你的数据，你可以通过设置菜单下载你自己的数据。

还有别的我该知道的东西吗？ 在鼠标悬浮时，按住 SHIFT 以就地固定住一个窗口。所有内容都有工具提示，而你作为 DM 将永恒。
